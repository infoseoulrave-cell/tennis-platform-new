# 라켓 상세 페이지 명세 초안

작성일: 2026-03-17
문서 목적: 라켓 상세 페이지의 정보 구조, 노출 우선순위, 콘텐츠 규칙을 정의한다.

## 1. 페이지 목표

- 사용자가 이 라켓의 성향을 10초 안에 이해하게 한다.
- 왜 추천되는지 납득하게 만든다.
- 추천 스트링과 그립까지 함께 제안한다.
- 판매처 클릭으로 자연스럽게 연결한다.

## 2. 상단 퍼스트뷰 구성

상단에는 아래 요소가 반드시 들어가야 한다.

- 고화질 히어로 이미지
- 브랜드 / 시리즈 / 모델명
- 한 줄 총평
- 5축 점수 카드
- 판매 가격대
- 구매 CTA

## 3. 섹션 구성 순서

### 1. 히어로 섹션

- 대표 이미지
- 대표 썸네일 묶음
- 모델명
- 요약 태그
- 가격대
- 구매 CTA

### 2. 5축 해석 섹션

- 직구력
- 컨트롤
- 스핀 잠재력
- 충격흡수
- 안정감

각 축에는 아래가 포함된다.

- 점수
- 짧은 설명
- 해석 툴팁

### 3. 추천 대상 섹션

- 이런 플레이어에게 추천
- 추천되지 않는 경우
- 주의 포인트

### 4. AI 세팅 추천 섹션

- 추천 스트링 2~3개
- 추천 장력 범위
- 추천 그립 1~2개
- 왜 이 조합이 맞는지

### 5. 스펙 섹션

- 무게
- 밸런스
- 헤드사이즈
- 빔 두께
- 패턴
- 강성
- 스윙웨이트
- 권장 텐션

### 6. 판매처 섹션

- 판매처명
- 가격
- 할인 여부
- 마지막 갱신 시각
- 구매 링크

### 7. 비슷한 라켓 섹션

- 같은 세그먼트 대안
- 더 파워 있는 대안
- 더 컨트롤 좋은 대안

### 8. 콘텐츠 연결 섹션

- 이 라켓 관련 가이드
- 시리즈 비교 콘텐츠
- 스트링 선택 가이드

## 4. 5축 점수 표현 규칙

- 막대형보다 짧은 수치 + 해석 문구 조합이 적절하다.
- `-5 ~ +5`를 그대로 노출한다.
- 색상은 강/중/약 정도만 보여주고, 과도한 히트맵은 피한다.

예시

- 직구력 `+4`
- 컨트롤 `+1`
- 스핀 잠재력 `+3`
- 충격흡수 `+2`
- 안정감 `+1`

## 5. 사진 운영 규칙

### 필수 컷

- 히어로 컷 1장
- 전체 프레임 컷 1장
- 프레임 디테일 컷 1장
- 그립/버트캡 컷 1장

### 권장 컷

- 45도 앵글
- 스트링베드 디테일
- 페인트 디테일

### 금지

- 배경이 과하게 지저분한 이미지
- 해상도 낮은 썸네일 확대
- 각도와 밝기가 들쭉날쭉한 이미지 혼합

## 6. 문장 톤 규칙

- 브랜드 광고처럼 쓰지 않는다.
- "폭발적", "압도적", "혁신적" 같은 과장어를 피한다.
- 장점만 쓰지 않고 주의 포인트도 함께 쓴다.
- 초보자도 이해 가능한 단어를 우선한다.

## 7. 구매 전환 규칙

- 상단과 판매처 섹션에 CTA를 반복 배치한다.
- 추천 스트링/그립 카드에도 구매 또는 자세히 보기 CTA를 둔다.
- 판매처 수가 2개 이상이면 가격 비교가 먼저 보이게 한다.

## 8. 모바일 우선 레이아웃

노출 우선순위

1. 이미지
2. 모델명과 한 줄 총평
3. 5축 점수
4. 구매 CTA
5. 추천 스트링/그립
6. 상세 스펙

## 10. MVP 완료 기준

- 5축 점수 모두 노출
- 한 줄 총평 존재
- 주의 포인트 존재
- 스트링 2개 이상 추천
- 그립 1개 이상 추천
- 판매처 1개 이상 연결
- 고화질 대표 이미지 확보
