# 초기 라켓 데이터 소스 매트릭스

작성일: 2026-03-17
문서 목적: 한국 판매 라켓 DB 구축을 위한 우선 소스와 역할을 정리한다.

## 1. 우선 원칙

- 공식 스펙은 브랜드/공식 유통사를 우선한다.
- 판매 가능 여부와 가격은 전문몰과 쇼핑 API로 보완한다.
- 이미지 사용 권한은 별도 확인한다.

## 2. 소스 매트릭스

## Tier 1. 공식/공식 유통

### HEAD Korea

- 역할: 공식 스펙, 제품 구조, 이미지 후보
- 우선순위: 높음
- 메모: 한국 유통 기준 제품 확인에 유리

### 요넥스몰

- 역할: 공식 스펙, 국내 SKU, 이미지 후보
- 우선순위: 높음
- 메모: 국내 판매 모델 기준성 높음

### Wilson Korea

- 역할: 시리즈 구조, 공식 카피, 이미지 후보
- 우선순위: 높음
- 메모: 신제품 반영 속도 확인 필요

### Dunlop Sports Korea

- 역할: 국내 공식 유통 확인, 이벤트/신제품 추적
- 우선순위: 중상

### Babolat 국내 공식 유통 채널

- 역할: 공식 스펙 보완, 국내 유통 SKU 확인
- 우선순위: 중상
- 메모: 공식몰/공식 유통 경로를 먼저 확정할 필요 있음

## Tier 2. 전문 판매처

### 국제스포츠

- 역할: 다브랜드 판매 여부 확인, 가격, 세일 상태
- 우선순위: 높음

### 스포닥

- 역할: 다브랜드 판매 여부, 해석형 분류 참고
- 우선순위: 높음

### 기타 전문몰

- 역할: 누락 모델 보완, 가격 비교
- 우선순위: 중간

## Tier 3. 플랫폼/API

### 네이버 쇼핑 검색 API

- 역할: 판매처 확장, 가격 비교 보조
- 우선순위: 중상

### 네이버 커머스 API

- 역할: 제휴 구조 기반 카탈로그/상품 보강
- 우선순위: 중간

## 3. 초기 브랜드 우선순위

### 우선 1차

### 2차 확장

- 기타 국내 판매 강세 브랜드

## 4. 수집 전략

### 1단계

- 브랜드별 시리즈 목록 확보
- 대표 인기 모델 목록 확보
- 국내 판매 여부 확인

### 2단계

- 공식 스펙 정규화
- 판매처 오퍼 연결
- 대표 이미지 확보

### 3단계

- AI 5축 점수 계산
- 스트링/그립 페어링 연결

## 5. 우선 수집 대상 수량

- 브랜드 6개
- 라켓 150~250개
- 인기 시리즈 우선
- 입문자/중급자 수요 모델 우선

## 6. 리스크

- 브랜드별 표기 방식 불일치
- 연식 구분 혼선
- 이미지 사용 권한 이슈
- `swing_weight`, `stiffness_ra` 누락 가능성

## 7. 바로 할 일

1. 브랜드별 공식 소스 URL 확정
2. 판매처별 수집 방식 확정
3. 필수 필드 충족률 조사
4. 대표 30개 라켓 샘플 목록 만들기
