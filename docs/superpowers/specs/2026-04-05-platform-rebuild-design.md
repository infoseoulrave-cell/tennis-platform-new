# 테니스 플랫폼 재구축 설계

작성일: 2026-04-05
업데이트: 2026-04-06
상태: 설계 확정

## 1. 배경

기존 racketlab 앱(app-pi-seven-15.vercel.app)의 구조와 기능을 Next.js 코드베이스로 재구축한다.
Paperclip이 원본 코드를 덮어써서 소스가 유실되었으나, 배포된 사이트가 살아있어 레퍼런스로 삼는다.
Paperclip이 만든 백엔드(DB 스키마, 시드 데이터, 추천 엔진)는 유지하고, 프론트엔드를 원본 racketlab 구조로 재구축하되 미니멀 프리미엄 톤으로 비주얼을 정리한다.

## 2. 기술 스택 (유지)

- Next.js 15 (App Router)
- Drizzle ORM + PostgreSQL
- Tailwind CSS 4
- TypeScript

## 3. 비주얼 방향

**미니멀 프리미엄** (기존 racketlab의 올리브/크림 톤에서 전환)
- 화이트/그레이 기반, 넉넉한 여백
- 타이포 중심, 고품질 이미지가 주인공
- 블랙(#111) 악센트, 과도한 색상 지양
- 라켓/선수 고품질 사진 적극 활용

## 5. 페이지별 설계 (racketlab 기준)

### 5-1. 홈

1. **히어로 캐러셀** (다크 배경) — 신제품 라켓 3개 로테이션, 각각 브랜드/모델명/한줄평/핵심 스탯 태그/스펙 배지, 라켓 고품질 이미지
2. **CTA 버튼** — "추천 받기" / "전체 라켓 보기"
3. **퀵 링크 카드 4개** — AI 추천 / 비교 / 라켓 DNA / 스트링 가이드
4. **인기 라켓 TOP 5** — 순위 리스트 (브랜드, 모델, 스펙 요약, 가격)
5. **코트 업데이트** — 최신 뉴스/팁 피드 (날짜, 태그, 제목, 설명)
6. **프로 선수 × 라켓 시너지** — 선수 사진 + 이름 + 플레이스타일 태그 + 사용 라켓 + 5축 바 차트 + 시너지 분석 텍스트 (남녀 각 약 9명)
7. **테니스 상식** — 카드형 팩트 5개
8. **스트링 가이드** — 폴리에스터/멀티필라멘트/내추럴 거트/하이브리드 타입 카드
9. **푸터** — 라켓/가이드/더보기 링크 + SNS + 법적 고지

### 5-2. 라켓 찾기 (`/rackets`)

- 전체 라켓 그리드/리스트
- 필터: 브랜드, 무게, 헤드사이즈, 가격, 5축 성향
- 정렬: 인기순, 가격순, 축별 정렬
- 카드: 라켓 이미지 + 브랜드 + 모델명 + 스펙 요약 + 가격

### 5-3. 라켓 상세 (`/rackets/[slug]`)

**레이아웃:** 좌측 라켓 이미지(대형) / 우측 정보

1. **헤더** — 브랜드 > 모델명 + 한 줄 총평
2. **스펙 배지** — 무게, 헤드사이즈, 패턴
3. **5축 스탯** — 레이더(오각형) 차트 + 수평 바 차트 콤보
4. **정직한 능력 분석** — 강점 설명 + "대가 있음" 트레이드오프 경고
5. **가격 + 구매 CTA** — 최저가 + "구매처 보기" 버튼
6. **탭 섹션:**
   - 상세 스펙 (헤드사이즈, 스트링 패턴, 강성, 길이, 프레임 두께)
   - 추천 대상
   - 스트링 & 그립
   - 구매처
   - 유사 라켓

### 5-4. AI 추천 (`/recommendation`)

기존 diagnosis 로직 재활용. 미니멀 프리미엄 UI.

### 5-5. 비교 (`/compare`)

- 최대 3개 라켓 비교
- 5축 레이더 차트 겹침
- 스펙 비교표

## 6. 디자인 개선 사항 (racketlab 대비)

### 비주얼 톤 변경
- 올리브/크림 배경 → 순수 화이트/그레이
- 히어로 캐러셀 다크 배경 유지하되 더 세련되게
- 전체적으로 여백 늘리고 타이포 정리

### 이미지 추가
- 라켓 고품질 프로덕트 사진 (현재 placeholder만 있음)
- 프로 선수 사진 (현재 이니셜 아바타만 있음)
- Unsplash/공식 이미지 소스 활용

### 유지할 것
- 페이지 구조와 섹션 순서
- 프로 선수 × 라켓 시너지 섹션 (racketlab의 핵심 차별화)
- 정직한 능력 분석 + 트레이드오프 패턴
- 5축 레이더 + 바 콤보 차트
- 모바일 하단 탭 내비게이션

## 7. 기존 코드 활용 계획

### 유지 (백엔드)
- `src/db/schema/` — catalog, recommendation, partner, events 스키마
