# MVP 백로그 초안

작성일: 2026-03-17

## P0. 서비스의 존재 이유를 만드는 기능

### 1. 라켓 카탈로그 구축

- 브랜드 6~8개 선정
- 라켓 150~250개 수집
- 기본 스펙 정규화
- 기능성 해석 점수 체계 반영
- 한국 판매처 링크 연결
- 대표 이미지 확보

완료 기준

- 사용자가 브랜드/무게/헤드사이즈/가격으로 필터할 수 있다.

### 2. 플레이스타일 진단

- 질문지 설계
- 응답값 구조화
- 추천 규칙 테이블 초안 작성
- 결과 페이지 설계
- 추천 스트링/그립 연결 규칙 설계

완료 기준

- 사용자가 3분 내 설문을 끝내고 라켓 3~5개를 추천받을 수 있다.

### 3. 라켓 상세와 비교

- 상세 페이지 설계
- 라켓 비교 기능
- 추천 이유/주의점 노출
- 기능성 점수 카드 노출
- 추천 스트링/그립 모듈 노출

완료 기준

- 사용자가 후보 라켓 간 차이를 이해하고 구매 링크까지 이동할 수 있다.

### 4. 운영자 검수 툴

- 상품 등록/수정
- 태그 수정
- 가격 오퍼 관리
- 단종 처리

완료 기준

- 운영자가 개발자 도움 없이 핵심 상품 데이터를 관리할 수 있다.

## P1. 전환율과 리텐션을 올리는 기능

### 5. 콘텐츠 허브

- 입문자 가이드
- 스펙 해설 글
- 추천 큐레이션 모음
- 라켓 해석법 소개 콘텐츠

### 6. 세일/신상품 모듈

- 신제품 배지
- 할인 중 상품 묶음
- 시즌 기획전

### 7. 저장 기능

- 찜
- 최근 본 라켓
- 비교 리스트 저장

## P2. 확장 기능

### 8. 테니스 공 카테고리

- 공 종류별 설명
- 용도별 추천
- 구매 연결

### 9. 웨어러블 카테고리

- 스마트워치/심박계 큐레이션
- 플랫폼 호환성 안내
- 구매 연결

### 10. 개인화 고도화

- 클릭/찜/구매 기반 추천 보정
- 후기 기반 신뢰도 보정
- 코치 추천 뱃지

## 즉시 착수 순서

1. DB 스키마 확정
2. 진단 질문지 확정
3. 관리자 운영 플로우 설계
4. 리스트/상세/비교 IA 설계
5. 초기 수집 브랜드와 판매처 확정
