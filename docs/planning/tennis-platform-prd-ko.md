# 테니스 장비 플랫폼 기획서

작성일: 2026-03-17
대상 시장: 대한민국
출시 형태: 웹앱 우선, 모바일 앱은 후속

## 1. 프로젝트 한 줄 정의

한국에서 판매 중인 테니스 라켓, 공, 웨어러블 장비를 한 곳에서 이해하고 비교하고 추천받고 구매까지 연결해 주는 테니스 장비 특화 커머스형 콘텐츠 플랫폼.

## 2. 왜 이 서비스가 필요한가

국내 테니스 장비 시장은 이미 활발하지만 구매 경험은 여전히 파편화돼 있다.

- 브랜드 공식몰은 자기 브랜드 중심이라 비교가 어렵다.
- 전문 쇼핑몰은 상품 수는 많지만 초보자에게는 스펙 이해가 어렵다.
- 네이버쇼핑은 가격 탐색에는 좋지만 플레이 스타일 관점의 해석이 약하다.
- 입문자와 성장형 동호인은 "내게 맞는 라켓"보다 "많이 보이는 라켓"을 사는 경우가 많다.

따라서 이 프로젝트의 핵심 기회는 단순 쇼핑몰이 아니라 다음 4가지를 한 번에 제공하는 것이다.

1. 라켓을 쉽게 이해하게 해주는 설명형 콘텐츠
2. 한국 유통 기준으로 신뢰 가능한 라켓 DB
3. 플레이 스타일 진단 기반 추천
4. 실제 구매 가능한 가격 및 판매처 연결

그리고 이 4가지를 하나로 묶는 제품적 핵심은 "우리만의 라켓 해석 체계"다.
이 해석 체계는 사람의 감상보다 AI의 정규화된 DB 해석 엔진을 중심으로 설계한다.

## 3. 비전과 포지셔닝

### 비전

"한국 테니스 플레이어가 장비를 가장 똑똑하게 선택하는 출발점"

### 포지셔닝

- 쿠팡/네이버/11번가: 범용 커머스
- 브랜드 공식몰: 단일 브랜드 카탈로그
- 전문몰: 판매 중심
- 이 서비스: 장비 이해 + 진단 + 추천 + 구매 연결

즉, 정보 서비스와 커머스를 함께 묶은 "장비 탐색 엔진" 포지션이 적합하다.

## 4. 핵심 타깃 사용자

### 1차 타깃: 입문자와 초중급 동호인

- 테니스를 시작한 지 0~24개월
- 라켓 스펙을 읽어도 차이를 잘 모름
- 코치, 동호회, 유튜브 추천에 강하게 영향 받음
- 실패 비용이 큰 첫 라켓 또는 두 번째 라켓 구매가 고민

### 2차 타깃: 성장형 장비 교체 수요층

- NTRP 2.5~4.0 수준의 일반 동호인
- 파워/컨트롤/스핀/팔 부담 등 구체적인 고민이 있음
- 현재 라켓에서 아쉬운 점이 분명함
- 브랜드를 넘나들며 비교하려는 니즈가 있음

### 3차 타깃: 장비 덕후 및 비교 소비층

- 시리즈별 차이, 연식 차이, 밸런스 차이에 민감
- 출시 소식, 세일 정보, 한정판에 관심이 많음
- 비교표와 필터, 고화질 이미지 활용도가 높음

## 5. 핵심 가치 제안

### 사용자 가치

- 스펙을 모르는 사람도 이해할 수 있다.
- 내 플레이 스타일에 맞는 라켓을 빠르게 찾을 수 있다.
- 한국에서 실제 구매 가능한 제품만 볼 수 있다.
- 가격과 판매처를 함께 확인할 수 있다.
- 직구력, 컨트롤, 충격흡수 같은 기능성을 우리 언어로 직관적으로 이해할 수 있다.
- 라켓과 함께 잘 맞는 스트링, 그립까지 한 번에 추천받을 수 있다.

### 비즈니스 가치

- 고관여 상품군이라 추천 품질이 매출에 직접 연결된다.
- 브랜드/판매처 제휴, 리드 제공, 어필리에이트, 광고가 가능하다.
- 라켓에서 시작해 공, 스트링, 슈즈, 웨어러블로 확장 가능하다.

## 6. MVP 범위

### MVP에 반드시 포함할 것

- 한국 유통 라켓 DB
- 라켓 해석 점수 카드
- 라켓 상세 페이지
- 라켓 비교 기능
- 필터/정렬 기능
- 플레이 스타일 진단 설문
- 추천 결과 페이지
- 판매처 링크 및 가격 정보
- 고화질 이미지 갤러리
- 추천 스트링/그립 모듈
- 기초 장비 콘텐츠

### MVP에서 제외하거나 최소화할 것

- 실시간 재고 연동 전면 자동화
- 중고 거래
- 커뮤니티 게시판
- 직접 결제/풀필먼트
- 웨어러블 실데이터 동기화
- 방대한 리뷰 시스템

웨어러블은 MVP에서 "상품 탐색/구매 연결"까지만 두고, 운동 데이터 연동은 후속 단계가 적절하다.

## 7. 서비스 구조

### 주요 메뉴

- 홈
- 라켓 찾기
- 플레이스타일 진단
- 라켓 비교
- 테니스 공
- 웨어러블
- 콘텐츠
- 세일/신상품

### 핵심 페이지

#### 홈

- 오늘의 추천 라켓
- 입문자 추천 큐레이션
- 스핀형/컨트롤형/팔 편한 라켓 모음
- 인기 브랜드 바로가기
- 최신 출시 및 할인 소식

#### 라켓 리스트

- 브랜드, 무게, 헤드사이즈, 밸런스, 패턴, 가격대 필터
- 입문자용, 여성 추천, 팔 편함, 스핀형 같은 해석형 필터
- 한국 판매처 기준 구매 가능 여부 표기

#### 라켓 상세

- 고화질 대표 이미지와 각도별 이미지
- 기능성 점수 카드
- 핵심 특성 요약
- 스펙 표
- 이런 플레이어에게 추천
- 주의 포인트
- 추천 스트링/그립
- 비슷한 라켓 비교
- 판매처/가격/구매 링크
- 시리즈 설명과 연식 차이

#### 플레이스타일 진단

- 구력
- 스윙 속도
- 선호: 파워/컨트롤/스핀/타구감/팔 부담 감소
- 플레이 패턴: 베이스라이너/올라운더/서브앤발리 성향
- 현재 라켓 불만족 포인트
- 예산

#### 추천 결과

- 추천 라켓 3~5개
- 추천 이유
- 타협 포인트
- 비슷하지만 다른 대안
- 공/스트링/진동방지/웨어러블 추천 연결

## 8. 추천 엔진 초안

### 입력값

- 실력 수준
- 신체 정보: 성별/연령대는 선택값, 팔 부담은 필수
- 플레이 스타일
- 선호 타구 특성
- 예산
- 기존 라켓 사용 경험

### 라켓 메타데이터

- 브랜드
- 시리즈
- 출시 연도
- 국내 판매 상태
- 정가, 실판매가 범위
- 무게
- 밸런스
- 헤드사이즈
- 길이
- 프레임 두께
- 스트링 패턴
- 강성
- 권장 사용자 레벨
- 직구력/컨트롤/스핀 잠재력/충격흡수/안정감 점수
- 추천 플레이 타입 태그
- 추천 스트링 프로필
- 추천 그립 프로필

### 추천 방식

1. 하드 필터
2. 규칙 기반 1차 추천
3. 운영자 큐레이션 가중치
4. 클릭/구매 데이터 기반 후속 개선

초기에는 규칙 기반이 가장 현실적이다. 이유 설명이 가능하고 운영팀이 품질을 빠르게 보정할 수 있기 때문이다.

### 예시 규칙

- 입문자 + 팔 부담 우려 + 예산 20만~30만원
  - 280g 전후, 100~105 sq.in, 중간 강성, 관용성 높은 프레임 우선
- 중급 + 스핀 지향 + 베이스라인 랠리 중심
  - 98~100 sq.in, 300g 전후, 스핀 계열 패턴과 에어로 성향 우선
- 컨트롤 지향 + 빠른 스윙 + 경기 지향
  - 98 sq.in, 305g 이상, 낮은 파워 대신 안정성과 방향성 우선

## 9. 카테고리 전략

### 1단계 핵심 카테고리

- 테니스 라켓

### 2단계 확장 카테고리

- 테니스 공
- 스트링
- 그립/댐프너/납테이프
- 라켓 가방

### 3단계 확장 카테고리

- 웨어러블
- 슈즈
- 의류

라켓이 서비스의 정체성이고, 공은 반복 구매 상품이라 리텐션에 유리하다. 웨어러블은 상징성은 크지만 데이터 연동 난이도가 높아 상품 큐레이션부터 진입하는 것이 좋다.

## 10. 웨어러블 전략

### MVP

- 스마트워치/심박계/운동 추적 웨어러블을 카테고리화해 탐색 및 구매 연결 제공
- 추천 기준은 테니스 적합성 중심으로 제시
  - 심박/활동 추적
  - 배터리
  - 워치 착용감
  - Android/iPhone 호환성
  - 운동 기록 생태계

### 후속 단계

- Apple Health, Samsung Health, Garmin 계열 연동 검토
- 플레이 기록과 장비 추천의 연결
  - 경기 빈도 높음 -> 내구성/충격 흡수 성향 강화
  - 활동량 높음 -> 회복/피로 관점 콘텐츠 연결

단, 웹앱만으로는 헬스 데이터 연동 경험이 제한적이므로 실데이터 연동은 모바일 앱 또는 모바일 브리지 전략 이후가 적합하다.

## 11. 콘텐츠 전략

상품 DB만으로는 차별화가 약하다. 검색 유입과 전환을 위해 설명형 콘텐츠가 필요하다.

### 추천 콘텐츠 주제

- 초보자가 첫 라켓 고르는 법
- 285g vs 300g 차이
- 헤드사이즈 98과 100의 차이
- 스핀형 라켓과 컨트롤형 라켓의 차이
- 팔이 아플 때 피해야 할 라켓 특성
- 테니스 공 종류와 상황별 선택법
- 테니스용 스마트워치 고르는 법

### 콘텐츠 형식

- 에디토리얼 기사
- 짧은 카드형 요약
- 라켓 비교표
- 입문자용 인터랙티브 가이드

## 12. 수익모델

### 1차

- 판매처 제휴 링크
- 리드/어필리에이트 수수료
- 브랜드 스폰서형 노출

### 2차

- 브랜드 런칭 페이지 제작
- 추천 결과 내 프로모션 슬롯
- 세일/신제품 알림 구독

### 3차

- 직접 판매 또는 멀티셀러 마켓플레이스
- 프리미엄 장비 추천 리포트
- 코치/샵 제휴 패키지

## 13. 운영 모델

### 콘텐츠 운영

- 장비 MD 또는 테니스 경험자 에디터 필요
- 추천 문구는 단순 번역이 아니라 한국 동호인 언어로 재해석

### 상품 운영

- 브랜드/판매처별 신규 출시 체크
- 단종/품절/가격 변동 관리
- 이미지 라이선스 관리

### 데이터 운영

- 정기 수집
- 중복 모델 정리
- 연식별 버전 구분
- 수동 QA

## 14. 차별화 포인트

### 차별화 1: 해석형 DB

단순 스펙 나열이 아니라 "어떤 사람에게 왜 맞는지"를 구조화한다.

### 차별화 1-1: 라켓 해석 언어

브랜드 문구를 옮기는 것이 아니라 직구력, 컨트롤, 충격흡수 같은 항목을 우리 기준으로 점수화해 한국 사용자에게 읽히는 언어로 설명한다.

### 차별화 1-2: AI 객관 해석 엔진

라켓을 AI가 동일 기준으로 분류하고 점수화하며, 설명과 스트링/그립 추천까지 일관되게 생성한다.

### 차별화 2: 한국 유통 기준 추천

해외 리뷰 중심이 아니라 한국에서 실제 구매 가능한 제품 중심으로 추천한다.

### 차별화 3: 이미지 중심 경험

고화질 사진, 각도별 이미지, 비교 시각화로 장비 탐색의 몰입도를 높인다.

### 차별화 4: 장비 연결 경험

라켓 추천 후 공, 스트링, 웨어러블까지 이어지는 번들 탐색 흐름을 만든다.

- 카테고리 확장에 따른 객단가 상승
- 브랜드/판매처 제휴 수

## 16. 6개월 로드맵

### 0~4주: 기초 설계

- 브랜드/판매처 소스 확정
- 라켓 DB 스키마 설계
- 추천 질문지 설계
- 콘텐츠 톤앤매너 정의

### 5~8주: 데이터 구축과 MVP 디자인

- 우선 브랜드 6~8개 라켓 데이터 수집
- 상세 페이지 구조 설계
- 리스트/비교/추천 UX 설계
- 이미지 및 라이선스 정책 정리

### 9~12주: MVP 개발

- 웹앱 프런트엔드
- 관리자용 상품 등록/검수 기능
- 추천 로직 1차 반영
- 가격/판매처 연결

### 13~16주: 소프트 런치

- 베타 사용자 피드백
- 추천 품질 보정
- 검색엔진 최적화
- 초기 콘텐츠 20~30개 확보

### 17~24주: 확장

- 공/스트링 카테고리 강화
- 웨어러블 카테고리 오픈
- 알림/찜/비교 저장 기능 추가

## 17. 주요 리스크와 대응

### 리스크 1: 이미지 저작권

- 대응: 브랜드/판매처와 제휴해 정식 제공 이미지 사용
- 비제휴 상태에서는 공개 사용 허용 범위를 엄격히 검토

### 리스크 2: 가격/재고 최신성

- 대응: 상세 페이지에 갱신 시점 표기
- 가격은 기준가와 최저가를 분리 표기

### 리스크 3: 추천 신뢰도

- 대응: 추천 이유와 비추천 이유를 함께 보여줌
- 코치/시타 리뷰 기반 보정 태그 운영

### 리스크 4: 초기 운영 리소스 부족

- 대응: MVP는 라켓 중심으로 좁게 시작
- 공/웨어러블은 카테고리 깊이를 제한

## 18. 지금 가장 중요한 결정

출시 초기에 가장 중요한 것은 "상품 수"보다 "추천 신뢰도"다.

따라서 MVP 원칙은 아래처럼 잡는 것이 좋다.

- 브랜드 수는 넓게, 데이터 품질은 더 깊게
- 많은 상품보다 대표 상품 우선
- 자동 수집보다 운영 검수 우선
- 직접 판매보다 구매 연결 우선
