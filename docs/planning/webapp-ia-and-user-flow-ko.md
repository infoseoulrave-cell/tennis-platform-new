# 웹앱 IA 및 사용자 흐름 초안

작성일: 2026-03-17
문서 목적: 테니스 장비 웹앱 MVP의 정보구조와 핵심 사용자 흐름을 정의한다.

## 1. IA 원칙

- 사용자는 로그인 없이도 라켓 탐색과 추천까지 경험할 수 있어야 한다.
- 홈에서 바로 "추천받기" 또는 "직접 찾기" 두 갈래로 진입할 수 있어야 한다.
- 모든 탐색의 끝은 라켓 상세와 구매 연결이어야 한다.
- 라켓 상세는 항상 스트링/그립 추천까지 포함해야 한다.

## 2. 최상위 메뉴

- 홈
- 라켓 찾기
- AI 추천
- 비교
- 콘텐츠
- 세일/신상품

보조 메뉴

- 공
- 웨어러블

- 최근 본 라켓

## 4. 핵심 사용자 흐름

## 4-1. 추천 중심 흐름

1. 홈 진입
2. "AI로 내 라켓 찾기" 클릭
3. 플레이스타일 설문 진행
4. 추천 결과 3~5개 확인
5. 라켓 상세 진입
6. 스트링/그립 추천 확인
7. 판매처 클릭

### 핵심 성공 기준

- 설문 완료까지 3분 이내
- 결과 페이지 이탈 없이 상세 진입
- 상세에서 구매 링크 클릭

## 4-2. 탐색 중심 흐름

1. 홈 또는 라켓 찾기 진입
2. 브랜드/무게/가격/5축 성향 필터 적용
3. 리스트 카드 탐색
4. 비교 담기
5. 비교 페이지 확인
6. 상세 진입
7. 구매 연결

## 4-3. 콘텐츠 유입 흐름

1. 검색엔진 또는 SNS로 콘텐츠 페이지 유입
2. 콘텐츠 중간 추천 라켓 카드 노출
3. 라켓 상세 진입
4. 추천 설문 또는 구매 연결

## 5. 화면별 역할

## 5-1. 홈

목적

- 처음 방문한 사용자에게 서비스 정체성을 즉시 전달
- 추천과 탐색 두 경로를 명확히 열어줌

핵심 섹션

- 메인 히어로
- 5축 해석 소개
- AI 추천 시작 CTA
- 입문자용 큐레이션
- 인기 라켓
- 시리즈별 추천
- 콘텐츠 피드

## 5-2. 라켓 찾기

목적

- 사용자가 직접 조건과 성향으로 탐색할 수 있게 함

필수 기능

- 필터
- 정렬
- 카드형 리스트
- 비교 담기
- 판매 가능 배지

권장 필터

- 브랜드
- 가격대
- 무게
- 헤드사이즈
- 직구력
- 컨트롤
- 스핀 잠재력
- 충격흡수
- 안정감
- 입문자 추천
- 팔 부담 낮음

## 5-3. AI 추천 시작

목적

- 질문 수를 최소화해 추천의 진입장벽을 낮춤

질문 원칙

- 한 화면 한 질문 또는 소그룹 질문
- 6~8문항 이내
- 전문 용어 최소화

## 5-4. 추천 결과

목적

- "왜 이 라켓이 나에게 맞는지"를 설득

핵심 섹션

- 요약 프로필
- 추천 라켓 3~5개
- 추천 이유
- 주의 포인트
- 대체 옵션
- 추천 스트링/그립 미리보기

## 5-5. 비교

목적

- 후보 라켓 사이의 차이를 빠르게 이해하게 함

핵심 요소

- 5축 비교
- 무게/헤드/RA/스윙웨이트 비교
- 추천 대상 차이
- 구매 링크 비교

## 5-6. 콘텐츠

목적

- 검색 유입 확보
- 초보자 교육
- 추천 신뢰도 강화

핵심 형식

- 가이드
- 비교 기사
- 추천 큐레이션

## 6. MVP 우선순위 화면

### 반드시 먼저 설계할 화면

- 홈
- 라켓 찾기
- 라켓 상세
- AI 추천 시작
- AI 추천 결과
- 비교

### 후순위 화면

- 찜
- 마이페이지
- 웨어러블 상세
- 세일 특집 페이지

## 7. 사용자가 가장 자주 만나는 CTA

- AI로 내 라켓 찾기
- 이 라켓 자세히 보기
- 비교에 담기
- 추천 스트링 보기
- 구매하러 가기

## 8. 로그인 전략

MVP에서는 비로그인 사용성 우선이 적절하다.

### 로그인 없이 가능한 것

- 탐색
- 비교
- 추천
- 판매처 이동

### 로그인 시 제공할 수 있는 것

- 찜 저장
- 최근 본 기록 동기화
- 추천 결과 저장

## 9. 모바일 우선 고려사항

- 카드 리스트에서 5축 모두 보여주지 않는다.
- 리스트에서는 강점 2개만 보여준다.
- 비교는 세로 카드 스택 + 가로 스크롤 표 조합이 적절하다.
- 상세 상단에는 사진, 5축, 구매 CTA가 먼저 보여야 한다.

## 10. 성공적인 MVP 경험 정의

사용자가 첫 방문 후 아래를 5분 안에 끝낼 수 있어야 한다.

1. 서비스가 무엇인지 이해
2. 내게 맞는 라켓 후보 3개 확보
3. 그중 하나의 상세에서 스트링/그립 추천까지 확인
4. 판매처로 이동
