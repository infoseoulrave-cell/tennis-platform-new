# 문서 저장 규칙

이 프로젝트의 기획, 조사, 설계, 결정, 작업 기록은 모두 로컬 폴더 `docs/` 아래에 저장한다.

## 폴더 구조

- `docs/planning/`
  - 서비스 기획서
  - MVP 백로그
  - 로드맵
- `docs/research/`
  - 시장 조사
  - 경쟁사 조사
  - 참고 자료
- `docs/specs/`
  - DB 설계
  - 추천 로직
  - 기능 명세
- `docs/logs/`
  - 세션 로그
  - 작업 메모
  - 진행 현황

## 운영 원칙

- 새로 진행하는 기획과 논의 결과는 가능한 한 문서로 남긴다.
- 중요한 결정은 근거와 함께 기록한다.
- 다음에 다시 시작할 수 있도록 현재 상태와 다음 액션을 항상 남긴다.
- 기존 문서는 덮어쓰기보다 갱신 기록이 남도록 업데이트한다.
