# 테니스플랫폼

한국 시장을 타깃으로 한 온라인 기반 테니스 장비 웹앱 프로젝트.

핵심 방향

- 한국에서 실제 판매되는 테니스 라켓 DB 구축
- AI 기반 5축 라켓 해석
- 플레이스타일 진단 기반 추천
- 추천 스트링/그립 연결
- 고화질 이미지 중심 탐색 경험
